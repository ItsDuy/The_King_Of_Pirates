Welcome to The King of Pirates, a strategic 2D platformer and base-defense game built in Unity! Inspired by the mechanics of King of Thieves, this project was developed as a personal one-month challenge to create a polished, functional game loop from scratch.

🎮 About the Project

The King of Pirates combines platforming precision with strategic base building. Players must navigate treacherous dungeons to steal gold and gems while designing their own layouts to protect their loot from rival pirates.

This project was developed using Unity 2022.3.44f1 with a target resolution of 16:9.

Key Features:

Precision Platforming: Navigate through traps, saws, and guards with tight, responsive 2D controls.

Trap Variety: Includes a range of classic obstacles: Cannons, Rotating Saws,Spike Heads, Spike Balls.


🛠️ Installation Instructions

To dive into the code or build the project locally, follow these steps:

Download Unity:

Ensure you have Unity version 2022.3.44f1 installed.

Available via Unity Hub.

Clone the Repository:

git clone [https://github.com/ItsDuy/the-king-of-pirates.git](https://github.com/ItsDuy/the-king-of-pirates.git)


Open the Project:

Launch Unity Hub.

Click Add -> Add project from disk and select the cloned folder.

Build and Run:

Go to File > Build Settings.

Select your platform (PC, Mac, or WebGL).

Click Build and Run.
